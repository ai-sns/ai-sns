## 常见问题排查

### Q1: 切换回Agent后仍提示"请先选择一个Agent"

**检查：**
1. 打开浏览器控制台，查看是否有错误
2. 检查 `window.agentState.currentAgentId` 是否有值：
   ```javascript
   console.log('Current Agent ID:', window.agentState?.currentAgentId)
   console.log('Agents List:', window.agentState?.agents)
   ```
3. 如果都是null/空，说明状态被意外清空了

**解决：**
- 刷新页面重新初始化
- 检查是否有其他代码调用了 `agentState.reset()`

### Q2: 侧边栏内容消失或不正确

**检查：**
1. 查看浏览器控制台的日志
2. 检查DOM中是否存在 `#sidebar-agent` 元素：
   ```javascript
   document.getElementById('sidebar-agent')
   ```

**解决：**
- 按Ctrl+R刷新页面
- 清除浏览器缓存后重启应用

### Q3: 切换很慢或卡顿

**检查：**
1. 查看浏览器控制台是否有大量重复渲染的日志
2. 打开Performance标签，录制切换操作，查看性能瓶颈

**解决：**
- 检查是否有无限循环的事件监听器
- 检查是否有内存泄漏
