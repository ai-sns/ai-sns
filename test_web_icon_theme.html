主题特性说明

  • 亮色主题：使用 Indigo 蓝色渐变 (#4F46E5 → #4338CA)

  • 自动适配：CSS 变量自动响应主题切换
  • 平滑过渡：0.3s 过渡动画，视觉体验流畅
