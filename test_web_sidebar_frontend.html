🧪 测试步骤

1. 测试搜索功能 ✓ 已实现

在应用中进入 Web 页面，测试以下操作：

  • 在 LLM Online 搜索框输入关键词
  • 验证列表实时过滤
  • 清空搜索框，验证显示所有项目
  • 在 AI Tools Online 区域重复测试

2. 测试编辑功能 ✓ 已实现

测试编辑现有项目：

  • 点击 "Manage" 按钮打开管理对话框
  • 点击某个项目的编辑按钮（铅笔图标）
  • 修改表单中的字段（Name、Title、URL、Description、Filename）
  • 点击 "Save" 保存
  • 验证管理对话框和侧边栏都已更新

3. 测试删除功能 ✓ 已实现

测试删除项目：

  • 点击 "Manage" 按钮打开管理对话框
  • 点击某个项目的删除按钮（垃圾桶图标）
  • 在确认对话框中点击确认
  • 验证项目已从列表中移除
  • 验证侧边栏也已更新

4. 测试拖拽排序功能 ✓ 已实现

测试拖拽改变顺序：

  • 点击 "Manage" 按钮打开管理对话框
  • 按住某个项目的拖拽手柄（左侧的六点图标）
  • 拖动到新位置
  • 释放鼠标
  • 验证位置已更新
  • 关闭对话框，验证侧边栏中的顺序也已更新
  • 刷新页面，验证顺序保持不变（已保存到数据库）

🔌 API 端点

GET /api/system/web-mng

获取所有 web 管理项目（按 position 排序）

POST /api/system/web-mng

创建新的 web 管理项目

PUT /api/system/web-mng/{item_id}

更新指定项目

DELETE /api/system/web-mng/{item_id}

软删除指定项目

PUT /api/system/web-mng/reorder

批量更新项目位置
